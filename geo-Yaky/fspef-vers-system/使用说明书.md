# FSPEF-VERS 频率共振直接勘探分析系统 — 使用说明书

---

## 一、系统概述

### 1.1 系统简介

本系统基于 **Yakymchuk 频率共振直接勘探技术**（FSPEF-VERS）理论开发，是一套 Web 端地质勘探分析平台。核心思想是：每种地下物质（石油、天然气、氢气、金矿、地下水）具有可识别的**特征共振频率**，通过短脉冲电磁场激发 → 地面接收频谱 → 参考光谱库匹配 → 频率识别物质身份 + 深度公式换算埋深，实现**直接定位矿体与流体**。

与传统的"找构造"（先找断层、圈闭，再推断含矿性）不同，本系统采用"**直接找物质**"范式——绕过构造解释，直接回答"地下有什么、在哪、多深"。

### 1.2 核心理论

| 概念 | 说明 |
|------|------|
| **Tesla 地球驻波理论** | 地球作为多层球形电容器，从地核到地表各层具有不同的介电常数和接触电位差，内部存在驻半波电磁场 |
| **物质特征共振** | 每种化学元素/化合物/矿物具有可识别的电磁共振频率"指纹" |
| **深度公式** | **H = C / (2f)**，其中 H=埋深，C=等效波速，f=共振频率。当探测频率的半波长等于目标深度时，驻波干涉最强 |
| **光谱匹配** | 将观测频谱与参考光谱库比对，通过余弦相似度/相关系数判定物质身份 |

### 1.3 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Python 3.13 + FastAPI + NumPy + SciPy + PyWavelets + PyKrige |
| 前端 | Vue 3 + TypeScript + Vite + Plotly.js + Three.js + Leaflet |
| 数据库 | SQLite + SQLAlchemy |

## 四、操作流程

### 4.1 流程总览

```
仪表盘 → 新建分析 → 选择目标物质 → 启动演示分析
                                        ↓
                              8步流水线自动运行
                              （实时进度显示）
                                        ↓
                              ┌──── 查看结果 ────┐
                              ↓         ↓         ↓
                          地图热力图  频谱分析   3D模型
```

### 4.2 步骤详解

#### 第一步：新建分析

1. 点击左侧导航 **"新建分析"**
2. 在"选择目标物质"区域，点击物质标签选择要探测的目标：
   - **石油/原油** — 碳氢化合物特征共振（0.5–5.0 Hz）
   - **天然气** — 甲烷特征共振（2.0–15.0 Hz）
   - **氢气** — 深部无机成因氢（5.0–25.0 Hz）
   - **金矿** — 金矿物特征共振（1.0–8.0 Hz）
   - **地下水/深层水** — 流体特征共振（0.3–3.0 Hz）
   - 可多选，选中标签会高亮显示边框
3. 点击 **"启动演示分析"** 按钮

#### 第二步：观察8步处理流水线

启动后，右侧面板实时显示8个阶段的执行状态：

| 阶段 | 名称 | 说明 |
|------|------|------|
| Stage 1 | 预处理（去噪/校正） | 高斯/中值滤波、基线校正、归一化 |
| Stage 2 | 频域变换（FFT/CWT） | 快速傅里叶变换，将时域信号转换到频率域 |
| Stage 3 | 特征提取（峰值/Q因子） | 检测共振峰频率、计算频带能量和品质因数 |
| Stage 4 | 光谱匹配（参考库对比） | 余弦相似度/相关系数匹配参考光谱 |
| Stage 5 | 物质识别（分类/置信度） | 阈值判定 + 置信度评分 |
| Stage 6 | 深度换算（H=C/2f） | 核心深度公式计算埋深，多频聚合 |
| Stage 7 | 异常分割（连通域分析） | 连通域分析 + 形态学滤波圈定异常区 |
| Stage 8 | 3D建模（Kriging/等值面） | 克里金插值 + MarchingCubes等值面提取 |

- ✓ 已完成（绿色）
- ⟳ 正在执行（蓝色高亮）
- ○ 等待中（灰色）

底部进度条显示总体百分比和当前消息。演示模式下，30×30网格通常在 **30–60秒** 内完成。

#### 第三步：查看分析结果

分析完成后，页面出现 **"查看分析结果 →"** 按钮，点击进入地图视图。

## 六、物质参数参考表

### 6.1 物质参数

| 物质 | 频率范围 (Hz) | 等效波速 C_eq (m/s) | 匹配阈值 | 典型深度 |
|------|-------------|---------------------|---------|---------|
| 石油/原油 | 0.5 – 5.0 | 300 | 0.75 | < 57 km |
| 天然气 | 2.0 – 15.0 | 250 | 0.75 | < 57 km |
| 氢气 | 5.0 – 25.0 | 200 | 0.70 | > 57 km |
| 金矿 | 1.0 – 8.0 | 350 | 0.70 | 变化大 |
| 地下水 | 0.3 – 3.0 | 280 | 0.72 | ~68 km |

### 6.2 深度计算示例

```
假设：测得石油共振频率 f = 1.2 Hz，等效波速 C = 300 m/s

深度 H = C / (2 × f)
       = 300 / (2 × 1.2)
       = 300 / 2.4
       = 125 米
```

### 6.3 置信度评分规则

置信度由三个因素加权计算：

| 因素 | 权重 | 说明 |
|------|------|------|
| 光谱匹配分数 | 60% | 余弦相似度 / 相关系数 |
| 信噪比 (SNR) | 20% | 峰值能量与背景噪声的比值 |
| 峰值品质因数 (Q) | 20% | 共振峰的尖锐程度 |

## 八、8步处理流水线技术细节

### Stage 1：预处理
- **算法**：小波阈值去噪（db4小波，soft阈值）/ Savitzky-Golay 滤波
- **操作**：基线校正（3阶多项式拟合）、归一化（最大值归一）
- **输入**：原始时域信号
- **输出**：去噪后的归一化信号

### Stage 2：频域变换
- **算法**：快速傅里叶变换（FFT），加汉宁窗
- **参数**：采样率 50 Hz，采样点 512
- **输入**：去噪信号
- **输出**：频率轴 + 幅度谱

### Stage 3：特征提取
- **算法**：峰值检测（scipy.signal.find_peaks）+ 频带能量积分 + Q因子估算
- **Q因子**：Q = f_peak / Δf（-3dB带宽）
- **输出**：特征向量 [峰值频率, Q因子, 频带能量]

### Stage 4：光谱匹配
- **算法**：余弦相似度
  ```
  sim(A, B) = (A · B) / (||A|| × ||B||)
  ```
- **操作**：将观测频谱与参考库中所有光谱插值到统一频率网格后逐一比对
- **输出**：每种物质的匹配评分 [0, 1]

### Stage 5：物质识别
- **置信度公式**：
  ```
  confidence = 0.6 × match_score + 0.2 × SNR_norm + 0.2 × Q_norm
  ```
- **输出**：每个测点的物质类型标签 + 置信度分数

### Stage 6：深度换算
- **核心公式**：**H = C / (2f)**
- **多频聚合**：按峰值幅度加权平均多个共振频率的深度计算结果
- **不确定度**：加权标准差
- **输出**：每个测点的深度值 + 不确定度

### Stage 7：异常分割
- **算法**：阈值分割 → 形态学开闭运算 → 连通域分析
- **参数**：置信度阈值 0.5，最小异常面积 3 个像素
- **输出**：异常多边形边界 + 统计信息（面积、中心坐标、深度范围）

### Stage 8：3D建模
- **插值**：反距离加权法（IDW）生成3D标量场
- **等值面**：Marching Cubes 算法提取三角网格
- **体积估算**：阈值以上体素统计
- **输出**：顶点 + 面片数组（供 Three.js 渲染）

## 九、常见问题

### Q1：分析任务运行多久？
演示模式（30×30 网格）通常在 **30–60 秒** 内完成。分析时间与网格点数成正比。

### Q2：如何解读置信度？
| 置信度范围 | 含义 |
|-----------|------|
| > 0.7 | 高置信度，强烈建议关注 |
| 0.5 – 0.7 | 中等置信度，可能有异常 |
| 0.3 – 0.5 | 低置信度，需结合其他证据 |
| < 0.3 | 背景噪声区域 |

### Q3：深度数值的可靠性如何？
深度由 H = C/(2f) 换算得到。由于等效波速 C 的取值依赖于区域地质条件，演示模式中使用经验值。实际应用中需根据已知井标定数据校准 C 值。

### Q4：为什么演示中检测到了非目标物质？
系统对所有5种物质的参考光谱库进行全量匹配。即使只选了"石油"和"天然气"，如果信号中存在与"水"或"金矿"参考谱相似的频率成分，也会被检测到。这是正常的光谱匹配行为。

### Q5：端口被占用怎么办？
```bash
# 查找占用进程
lsof -ti:8000   # 后端端口
lsof -ti:5173   # 前端端口

# 强制释放
kill -9 $(lsof -ti:8000)
kill -9 $(lsof -ti:5173)

# 或直接重启
./start.sh restart
```

### Q6：守护进程是如何工作的？
Watchdog 每 30 秒通过 HTTP 请求检测 Backend（`:8000/api/health`）和 Frontend（`:5173/`）的健康状态。如果任一服务无响应，Watchdog 会自动重新启动该服务并将事件记录到 `.logs/watchdog.log`。

### Q7：如何查看日志？
```bash
# 查看后端日志（API 请求、错误信息）
tail -f .logs/backend.log

# 查看前端日志
tail -f .logs/frontend.log

# 查看守护进程日志（自动重启记录）
tail -f .logs/watchdog.log
```
